О приложении

BatchConv Mobile предназначен для быстрой локальной обработки файлов прямо на телефоне без интернета и серверов.

Главная идея проекта — сделать простой и конфиденциальный инструмент, который группирует и конвертирует видео, аудио, изображения и документы за пару тапов.

  1. Пользователь выбирает один или несколько файлов.
  2. Приложение мгновенно определяет тип файлов.
  3. Пользователь выбирает целевой формат конвертации.
  4. Файлы последовательно обрабатываются локально.
  5. Результат автоматически сортируется по расширениям.

Python Core

Управляет очередью задач, вызывает FFmpeg для медиафайлов, Pillow для изображений, производит сканирование и сортировку готовых файлов на устройстве.

HTML5 / Tailwind CSS / JS

Обеспечивает современный адаптивный интерфейс с мгновенным откликом, нативной мобильной навигацией, переключением тем и локализацией.
